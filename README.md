# Nodejs Jest Api

Api with test for each endpoint.

## How to run
1. Clone the repo
2. Run `npm i`
3. Run `npm i -D`
4. Create an `.env` file and add `MONGO_URI`
5. Run `npm run test`

**The MONGO_URI var need to be a TEST db, because note.test.js erased all the db content before start with the test**
